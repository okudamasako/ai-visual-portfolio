# AI Visual Portfolio

AIを活用して制作したビジュアル作品を掲載した、
シンプルなポートフォリオサイトです。

## Overview

- レスポンシブ対応
- 画像グリッド表示
- クリック拡大表示（Lightbox）
- シンプルで清潔感のあるデザイン

## Tech Stack

- HTML
- CSS
- Vanilla JavaScript

## Usage

index.html をブラウザで開くだけで閲覧できます。

## Files

- index.html
- image files (.png)

## Purpose

クラウドワークス等の応募時に、
ポートフォリオURLとして提出する用途を想定しています。
